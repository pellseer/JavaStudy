== 작성규칙

** 'Java 웹 개발 마스터 올인원 패키지 Online 강좌 중 Java'의 코드를 작성한다.
** 추가적으로 설명이 필요한 부분이나 중요한 내용은 노션에 정리한 후 링크를 걸어준다.
** 코드는 직접 작성해야만 기억에 남는다는 생각으로 작성한다.
** 빨리 끝내는게 중요한게 아니라 완벽히 내 것으로 만드는게 중요하다.
